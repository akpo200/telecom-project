## 3. YAS MONEY (Mobile Money)

YAS Money permet d'envoyer de l'argent, payer des factures et acheter du crédit.

**Codes USSD :**
*   Menu principal : `#144#`
*   Consulter solde : `#144*1#`
*   Envoyer argent : `#144*2*NUMERO*MONTANT*CODE#`

**Frais de retrait :**
*   0 - 5 000 : 100 FCFA
*   5 001 - 10 000 : 200 FCFA
*   10 001 - 50 000 : 1%
*   Plus de 50 000 : 0.8%

**Problème de transfert ?**
Si vous avez envoyé de l'argent au mauvais numéro, composez immédiatement le `#144*9#` pour tenter une annulation (faisable si le destinataire n'a pas encore retiré les fonds). Sinon, contactez le 200.

## 4. ASSISTANCE TECHNIQUE & CONFIGURATION

### 4.1 Configuration Internet (APN)
Si vous n'avez pas accès à internet, vérifiez vos paramètres APN.

*   **Nom** : YAS Internet
*   **APN** : yas.sn
*   **Nom d'utilisateur** : (vide)
*   **Mot de passe** : (vide)
*   **Type d'APN** : default,supl

Redémarrez votre téléphone après la configuration.

### 4.2 Problèmes de Réseau
1.  Activez puis désactivez le mode avion.
2.  Vérifiez que les données mobiles sont activées.
3.  Vérifiez votre couverture (la 5G nécessite une zone couverte et un téléphone compatible).
4.  Testez la carte SIM dans un autre téléphone.

### 4.3 Code PUK bloqué
Si vous avez bloqué votre code PIN (3 essais erronés), le code PUK est requis.
Il est disponible :
*   Sur le support de votre carte SIM.
*   En appelant le service client au 200 (vérification d'identité requise).
*   Via l'application "YAS & Moi".

*Attention : 10 essais erronés de code PUK grillent définitivement la carte SIM.*
